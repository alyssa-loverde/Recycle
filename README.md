# Recycle

Our website, **Recycle!**, aims to connect users with nearby recycling centers, make users aware about common items, and promote recycling by having a leaderboard.

## Features
- Based on the user's location, identify the nearest recycling center.
- Has a database of common items and recycling codes, which users can contribute to.
- Has a leaderboard - users can give themselves a point each time they give something for recycling

## Getting Started
1. Clone the repository
- git clone https://github.com/alyssa-loverde/Recycle
- cd Recycle
2. Set up the Backend
- cd backend
- pip install flask flask-cors
- python3 app.py
3. Set up the Frontend
- cd frontend
- npm install
- npm start

## Tech Stack
### Languages
- Python (backend)
- HTML, CSS, JavaScript (frontend)
### Tools and Frameworks
- SQLITE3 (user database)
- Flask (backend)
- Node.JS and React (frontend)
